Get Started with Docker

We have a complete container solution for you - no
matter who you are and where you are on your
containerization journey. 






    
OUTPUT
Docker Commands Description
docker ps -a show a list of all running containers (argument "-a" can be eliminated)
docker run IMAGE start a new container from a image
docker run --name CONTAINER IMAGE start a new container from a image and assign it a name
docker rm CONTAINER delete a container
docker start CONTAINER start a running container
docker stop CONTAINER stop a running container
docker cp CONTAINER:SOURCE TARGET copy a file from a container to the host
docker exec -it CONTAINER EXECUTABLE start a shell inside a running container
docker rename OLD_NAME NEW_NAME rename a container
docker commit CONTAINER create an image out of container
docker PULL IMAGE[:TAG] download an image
docker push IMAGE upload an image to a repository
docker images show list of all images
docker stats show stats of running containers
